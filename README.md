# Mobicam (free)

Public download repo for [Mobicam](https://github.com/alexsoh/mobicam) — an Android RTSP camera app for eVaultex (`io.evaultex.app.mobicam`).

## Install

Download the latest APK from **[Releases](https://github.com/alexsoh/mobicam.free/releases)**.

Releases are published automatically when a new version is shipped from the main Mobicam repo.

## Requirements

- Android 7.0+ (API 24)
- Camera and microphone permissions for live streaming
